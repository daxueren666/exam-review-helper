# 第三章 数据结构基础

本章介绍计算机科学中最基础的数据结构，包括数组、链表、栈、队列。

## 3.1 数组

数组是一种线性数据结构，通过索引访问元素。数组的特点是随机访问 O(1)，但插入删除 O(n)。

### 3.1.1 一维数组

一维数组是最简单的数组形式。在内存中连续存储。

### 3.1.2 二维数组

二维数组可以看作数组的数组，常用于表示矩阵。

## 3.2 链表

链表是一种动态数据结构，通过指针连接节点。链表的特点是插入删除 O(1)，但随机访问 O(n)。

### 3.2.1 单链表

单链表的每个节点包含数据和指向下一个节点的指针。

### 3.2.2 双链表

双链表的每个节点包含数据、指向前驱和后继的指针。

## 3.3 栈

栈是一种后进先出（LIFO）的数据结构。基本操作：push、pop、peek。

## 3.4 队列

队列是一种先进先出（FIFO）的数据结构。基本操作：enqueue、dequeue。
